# spotipaper
Authenticates with Spotify API and sets background to whatever track is playing.

Instructions (macOS):
```
python3 -m venv venv
source venv/bin/activate #if in vs code, agree to the dialogue
pip install -r requirements.txt 

python3 flask_server.py
```
